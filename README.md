# Formulario-Metodos_HTTP-Consumo_JSON
formulário consumindo arquivo JSON
<p>Projeto feito com uso dos metodos HTTP (POST, GET, PUT, DELETE), consumindo API de arquivo JSON online (no github) e em localhost com json-server</p>
<p>Para visualizar esse projeto é necessario Node.js<p>
<p>Faça download da pasta "projeto" e abra o terminal (CMD) dentro da pasta "public", em seguida use o comando:</p>

```
npx json-server db.json
```

<p>e abra o arquivo "index.html"</p>

<p>o json server permitirá o consumo dos arquivos .json e dará retorno sobre as requisições realizadas</p>
